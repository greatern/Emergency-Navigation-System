# Graph-Based Emergency Navigation System | Python, Dijkstra’s/A Algorithms,kNN, RAG*

A system that analyzes building floor plans to find optimal evacuation routes and
classify risk zones.
• Algorithm Implementation: Developed Dijkstra’s and A* algorithms to calculate
shortest evacuation paths, reducing simulated escape time by 35%.
• Image Classification: Applied Region Adjacency Graphs (RAG) and k-NN to classify
high-risk zones in floor plans with 90% accuracy.

• Team Collaboration: Partnered with 3 peers to design graph structures
nodes=rooms, edges=pathways and a GUI for real-time visualization.
